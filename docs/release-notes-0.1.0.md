## MetaDrop 0.1.0

The first native Linux release establishes the complete inspect → sanitize → verify workflow.

### Highlights

- C++20 and Qt 6 Widgets interface that follows the desktop theme.
- Batch drag-and-drop, system tray, detailed metadata tree, risk classification, and automatic-copy mode.
- Verified cleaning for image metadata, audio tags, PDF document metadata, and OOXML/OpenDocument properties.
- Non-destructive output, owner-only staging files, atomic installation, and mandatory post-clean rescan.
- English and Russian interface resources.

### Download

- `MetaDrop-0.1.0-x86_64.AppImage` — portable Linux build.
- `metadrop_0.1.0_amd64.deb` — Debian/Ubuntu package.
- `MetaDrop-0.1.0-source.tar.gz` — release source.
- `SHA256SUMS` — integrity checks for every artifact.

Read the [verified format scope](https://github.com/Trendorin/MetaDrop#verified-format-scope) before using MetaDrop with sensitive material. Metadata removal reduces accidental disclosure but does not anonymize visible content or the account used to share it.
