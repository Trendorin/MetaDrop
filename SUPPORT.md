# Support

- Bugs and reproducible compatibility problems: [issue tracker](https://github.com/Trendorin/MetaDrop/issues/new/choose)
- Security vulnerabilities: [private security advisory](https://github.com/Trendorin/MetaDrop/security/advisories/new)
- Format support requests: [feature request](https://github.com/Trendorin/MetaDrop/issues/new?template=feature.yml)

When reporting a file-processing problem, use a synthetic sample with no personal or confidential data. Remove usernames, local paths, tokens and unrelated metadata from logs and screenshots.

MetaDrop is a community open-source project without a guaranteed support SLA.
